# Python-Math-Chat-Bot---Loop-based
A simple Python chatbot to perform mathematical calculations. It is a beginner friendly loop based chat project.


It can:
Say hello and answer simple questions
Solve math problems like addition, subtraction, multiplication, division, and parentheses
Remember past messages during the chat
Give a default reply if it doesn’t understand

To run it:
1. Download this repository.
2. Open the project folder.
3. Run `Math_Chat_Agent_using_Loop.ipynb` in your terminal.
4. Type messages and chat. Type "exit" to stop.
